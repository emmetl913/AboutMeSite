Sort Visualizer

Worked with a team to design a desktop application that visualizes various sorting algorithms. The application allows users to select different sorting algorithms and visualize the sorting process in real-time. The application also includes parallel processing support, allowing the user to specify the number of CPU cores to assign to the task, and watch them all execute together, in real time.
