YCP Hacks 2024

For my team project, me and three other teammates designed and implemented a web application that allows users to log and view analytics about their bowling matches. The application allows users to create and manage events, and log matches for each events. The application also allows users to view analytics about their sessions, such as their average score, and the number of strikes and spares they have bowled.

This included implementing both a front-end web page, as well as the server and SQL database to store the data.

For my individual project, I created a 2D game engine along with two games using the C# .NET framework Monogame. The engine was designed to be semi-modular, allowing for future implementation of new features. The main features of the engine include a tilemap system, a sprite animation system, and a camera system.
